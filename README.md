# TextEditor

## Описание

Программа умеет:

  * открывать файл;
  * сохранять файл;
  * закрываться;
  * читать и записывать в юникоде.
  
Программа написана с использованием:

  * PyQt4;
  * Qt Designer;
  * Pycharm.
